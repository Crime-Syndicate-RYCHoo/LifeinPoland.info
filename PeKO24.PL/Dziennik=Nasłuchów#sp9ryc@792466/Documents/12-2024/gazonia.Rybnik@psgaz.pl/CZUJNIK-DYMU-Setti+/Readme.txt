- How to install IrfanView languages?

  1) Find your IrfanView directory

  2) Create in the IrfanView directory a new directory called "Languages"

  3) Unzip all files from the language ZIP into the "Languages" directory

  4) Start IrfanView, go to menu: "Options->Change Language" and 
     choose your language ;-)



Much fun!
